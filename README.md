# oxidec

Eye-candy manager written in Rust

## Install

```sh
cargo install oxidec
```

**NOTE**: recommended aliases

```sh
alias cs="oxidec colorscheme"
alias wl="oxidec wallpaper"
alias wp="oxidec wallpaper"
# alias th="oxidec theme"
```
